# Global Book Library Website

A simple static website for browsing books, viewing full details, and simulating purchases from anywhere.

## Files

- `index.html` — website UI and page structure
- `styles.css` — responsive styling
- `script.js` — book data, search, cart, and checkout logic

## How to use

1. Open `index.html` in your browser.
2. Search books by title, author, category, or ISBN.
3. Click `View Details` for complete information.
4. Click `Buy Now` to add books to your cart.
5. Click `Checkout` to complete your order.

> This website is a static prototype. You can host it on any web server or platform such as GitHub Pages.
